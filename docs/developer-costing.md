## Proposal Tips

1. **Start with MVP only** - Easier for client to commit, builds trust
2. **Include buffer** - First-time software clients often have scope creep
3. **Milestone payments** - Protects both parties
4. **Written contract** - Clear deliverables, timeline, payment terms
5. **Change request process** - Define how to handle new requirements
